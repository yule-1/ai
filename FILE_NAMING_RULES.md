# 자료 저장 규칙

앞으로 생성하는 모든 자료는 아래 규칙을 따른다.

1) 저장 위치
- 기본 루트: `ai/`
- 요청 작업일 폴더: `ai/YYYYMMDD/`
- 예: `ai/20260220/`

2) 파일명 규칙
- 내용이 바로 드러나게 작성
- 영어 소문자 + 숫자 + 언더스코어 사용 권장
- 예시
  - `xovr_1y_volume_price_with_aum.png`
  - `xovr_1y_volume_price_with_aum.csv`
  - `fadu_vs_marvell_index100_2y.png`

3) 생성 원칙
- 파일은 해당 날짜 폴더에서 생성/저장
- 같은 주제 재생성 시 버전/시간 접미사 사용 가능
  - 예: `_v2`, `_20260220_1345`
